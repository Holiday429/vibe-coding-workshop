Vibe Check First:

Don't start with code; start with a conversation. Imagine you are a boss giving a brief to your AI colleagues — be structured and clear about what you expect.

The Structured Prompt Template

"You are an expert Frontend Developer and UI/UX Designer. You specialize in creating modern, responsive, and high-end personal portfolios using HTML, Tailwind CSS, and Lucide Icons."

"I want to build a personal website that acts as my [digital business card / project showcase / job-seeking portfolio]. The goal is to make a lasting impression on [recruiters / potential collaborators]."

Visual style: [Minimalist White / Cyberpunk Dark / Warm & Academic / High-tech Professional]
Color palette: [e.g., Deep navy and gold / All white with soft shadows]
Typography: [e.g., Elegant Serif for headings]
Reference: "I want it to feel like Apple product pages — lots of whitespace and smooth transitions."

Hero: big catchy headline + short bio
Project Gallery: 3–4 cards with images and descriptions
Tech Stack: clean grid showing skills
Contact: social links and email button

"Make it fully mobile-responsive. Keep the code modular and clean — separate the HTML, CSS, and JS so it's easy to manage."

Polish Checklist

🎨 UI Consistency
  • Are all button corners the same?
  • Is spacing between sections even?
  • Is the color palette unified?
"Review the entire page for UI consistency. Ensure all buttons use the same corner radius and padding. Make sure headings follow the same font weight and color scheme."
📱 Mobile Responsiveness
  • Does the text overlap on a small screen?
  • Do images scale down?
  • Is there a working mobile menu?
"Check the layout for mobile devices. Ensure the hero section doesn't overflow and the project grid stacks vertically on smaller screens. Add a responsive hamburger menu if missing."
👁️ Readability & Accessibility
  • Is there enough contrast between text and background?
  • Is the font size easy to read?
"Improve text contrast for better readability. If the background is dark, ensure body text is bright enough. Add 'alt' tags to all images for SEO and accessibility."
✨ Interactive Feedback
  • Do buttons react when you hover?
  • Do links actually go somewhere?
"Add subtle hover effects (slight lift or color shift) to all clickable cards and buttons. Ensure all external links open in a new tab (target='_blank')."
⚡ Assets & Performance
  • Are the images loading quickly?
  • Are any icons broken or missing?
"Optimize the page for speed. If I am using high-resolution images, suggest a way to lazy-load them. Also, verify that all icons (Lucide/FontAwesome) are rendering correctly."

Pro Tips for "Talk-to-Fix"

01

Don't Start Over

The AI learns from your conversation context. Say: "The button turned red, but I wanted blue. Fix that."

02

Screenshot & Feedback

Take a screenshot of the broken part and upload it directly to Claude for visual feedback.

03

One-at-a-Time Rule

Don't ask for 10 changes in one prompt — the AI gets confused. Ask 1–2, see the result, then continue.

1
Create a hosting account

Sign up on your preferred hosting platform.

2
Create a new site project

Create a project with a simple name (e.g. my-portfolio). Keep it public if required by your host.

3
Upload your files

Upload your HTML, CSS, and JS files in the project dashboard.

⚠️ Your main file MUST be named index.html

Save or publish the upload in your platform dashboard.

4
Enable web publishing

Open your host settings and enable static site publishing for this project.

★
Get your live link!

Wait ~1–2 minutes, then refresh. Your site will be live at:

https://your-site.com
